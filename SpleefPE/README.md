# SpleefPE
Plugin for Spleef to version 1.9 - 1.16 of Minecraft Bedrock Edition. The plugin adds a mini-game Spleef, a fun mini-game.

# Usage
1) Download the plugin.
2) Upload it to the plugins.
3) Restarting server.
4) Go to server.
5) Commands: /spleef
